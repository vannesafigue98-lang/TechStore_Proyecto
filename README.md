# TechStore Proyecto

## Descripción

Proyecto de tienda virtual desarrollado con HTML, CSS, Bootstrap y JavaScript.

## Funciones

* Página principal
* Navegación entre páginas
* Formulario de contacto
* Página de ayuda
* Diseño responsive

## Tecnologías utilizadas

* HTML5
* CSS3
* Bootstrap 5
* JavaScript
* Git y GitHub

## Ejecución

1. Abrir el proyecto en Visual Studio Code.
2. Ejecutar con Live Server.
3. Navegar entre las páginas del sitio.
