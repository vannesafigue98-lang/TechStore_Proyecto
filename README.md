3. Descripción de la tienda

TechStore es una tienda virtual enfocada en la venta de productos tecnológicos como audífonos, dispositivos electrónicos y accesorios. El sitio permite visualizar productos, navegar entre páginas y simular una experiencia básica de compra.

4. Objetivo del proyecto

Desarrollar una tienda virtual funcional utilizando tecnologías web básicas (HTML, CSS, Bootstrap y JavaScript), aplicando conocimientos de frontend, diseño responsive y manejo de interfaces interactivas.

7. Instrucciones de instalación y ejecución

Clonar el repositorio desde GitHub:

git clone <URL-del-repositorio>
Abrir la carpeta del proyecto en Visual Studio Code.
Instalar la extensión Live Server (si no la tienes).
Ejecutar el archivo index.html con Live Server.
Navegar por el sitio web.

8. Funcionalidades implementadas
Página principal con presentación de la tienda
Catálogo de productos
Formulario de contacto
Página de ayuda
Diseño responsive con Bootstrap
Interacción con JavaScript (manejo de productos)

9. Flujo de información
El usuario ingresa a la página principal.
Navega a la sección de productos.
Visualiza información de cada producto.
Puede enviar mensajes mediante el formulario de contacto.
Los datos del formulario se procesan con JavaScript (y backend si aplica en tu proyecto).
